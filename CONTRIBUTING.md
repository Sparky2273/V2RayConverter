# Contributing to V2Ray Converter

Thank you for your interest in contributing! Here is how you can help.

## 🐛 Reporting Bugs

Open an issue on the [Issues tab](../../issues) and include:

- Your operating system (Windows 10/11, Ubuntu 22.04, etc.)
- How to reproduce the bug step by step
- What you expected to happen vs. what actually happened
- Any error message you see (screenshot or paste the text)
- The input config or URI that caused the problem (remove any real passwords/UUIDs first)

## 💡 Suggesting Features

Open an issue describing:
- What the feature does
- Why it would be useful
- An example of how it would work

## 🔧 Pull Requests

1. Click **Fork** at the top of the repository page to create your own copy
2. Make your changes in your fork
3. Test that the app still runs correctly (`python V2RayConverter_V1.py`)
4. Open a **Pull Request** from your fork back to this repository
5. Describe what you changed and why

### Code Style
- Keep to the single-file structure if the change is small
- Follow the existing code style (type hints, docstrings, section comments)
- Test with at least one VLESS, VMess, Trojan, and Shadowsocks config

## 🌍 Translations

The app UI is in English. If you can translate the README into Persian (Farsi), Arabic, Russian, Chinese, or any other language — please contribute! Create a file named `README.fa.md`, `README.ar.md`, etc.
